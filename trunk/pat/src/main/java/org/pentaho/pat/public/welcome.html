Welcome

The PAT Analysis Tool is an Open Source OLAP Viewer that aims to give business users the ability to visualise their data in an easy to understand fashion

The PAT Analysis Tool also aims to be a framework that will run both standalone and integrated into a Web App, for example Pentaho BI Server with the ability to utilise the parent applications' data sources and security.

Version 1.0 isn't too far away and we're constantly looking for Beta Testers, CSS Designers, Translators and Programmers, if you are interested in helping please get in contact. We have a number of resources to help testers and end users:
