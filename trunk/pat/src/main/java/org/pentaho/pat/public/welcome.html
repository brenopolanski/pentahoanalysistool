PAT Analysis Tool

The PAT Analysis Tool is an Open Source OLAP Viewer that aims to give business users the ability to visualise their data in an easy to understand fashion.

What's New In 0.7?

  • New menubar + therefore a new workflow
  • drillthrough on mondrian connections
  • initial property support (discovery)
  • improved connection workflow
  • fix bug in excel export
  • various minor bugfixes and improvements
